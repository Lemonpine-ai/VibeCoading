## 플로우 화살표(MCP) 한계 — 해결 방법

Talk to Figma MCP의 `create_connections`는 **이미 파일 안에 “기본으로 쓸 커넥터(Connector)” 노드가 있어야** 동작합니다. 처음에는 없어서 오류가 납니다.

### 방법 A — FigJam 커넥터 복사 (MCP로 자동 연결까지 쓰고 싶을 때)

1. **FigJam** 새 파일을 엽니다.
2. 도구에서 **Connector**(화살표)를 하나 그립니다. 스타일·두께는 원하는 대로.
3. 그 커넥터를 **복사(Ctrl/Cmd+C)** 합니다.
4. **다보냥이 있는 Figma 디자인 파일** 같은 페이지에 **붙여넣기(Ctrl/Cmd+V)** 합니다.
5. Cursor에서 Talk to Figma MCP로 **`set_default_connector`** 를 실행할 때, **방금 붙여 넣은 커넥터의 노드 ID**를 넘깁니다.  
   - 플러그인/선택으로 ID 확인 후 MCP에 전달.
6. 그 다음 **`create_connections`** 에 `startNodeId` / `endNodeId` 를 넣으면, 같은 스타일로 프레임 간 연결이 생성됩니다.

### 방법 B — Figma 디자인 파일에서만 수동 (가장 단순)

1. Figma **디자인** 모드에서 **리소스(Shift+I)** 또는 상단 **FigJam** 섹션에서 **Connector**를 씁니다. (없으면 FigJam 파일에서 복사해 오는 A와 동일)
2. 프레임 `3:22` → `3:28` → `4:48` → `4:49` 순으로 **드래그해 연결**합니다.
3. 라벨은 커넥터 선택 후 텍스트 추가.

### 방법 C — 프로토타입 모드

1. **Prototype** 탭에서 각 버튼·탭에 **인터랙션**을 연결해 화면 전환을 만듭니다.  
2. “화살표”는 **플로우 다이어그램**이 아니라 **클릭 플로우**로 검증할 때 유리합니다.

**정리:** MCP로 화살표를 자동으로 쓰려면 **한 번은 커넥터 노드를 페이지에 넣고 `set_default_connector`로 등록**해야 하고, 그게 번거로우면 **방법 B로 수동 연결**하면 됩니다.

권장 연결(참고):
1. `3:22` → `3:28` (시작하기)
2. `3:28` → `4:48` (Health 탭)
3. `4:48` → `4:49` (Account 탭)

30년차 시니어 톤으로 **여백·대비·타이포**를 우선합니다. 장식(그라데이션 남발, 파스텔 핑크)은 줄이고 **모노크롬 + 단일 액센트**로 통일합니다.

## CATvisor 1–4를 더 세련되게 + **다크 버전**까지 (여기서/MCP)

### Cursor에서 **가능한 조건**
Talk to Figma MCP는 **Figma 데스크톱 + 플러그인**이 켜져 있고, **소켓**(`cursor-talk-to-figma-socket`)이 실행 중이며, Cursor와 **같은 채널**(현재 **`68ck883i`**)로 조인되어 있어야 노드 조회·색 변경·복제 등이 동작합니다. 연결이 없으면 MCP 요청이 **타임아웃**됩니다. → **지금은 Figma 쪽 연결을 켠 뒤** 같은 채널로 다시 시도하면 됩니다.

### 다크 세트를 만드는 표준 방법 (Figma에서 2분)
1. 프레임 `3:22`, `3:28`, `4:48`, `4:49` 를 **한 번에 선택**합니다.  
2. **⌘/Ctrl+D** 로 복제합니다.  
3. 복제본을 **아래로** `≈920px` 정도 이동해 기존 라이트/틸 버전과 겹치지 않게 둡니다.  
4. 레이어 이름을 `(다보냥 · Dark) 01 …` 형식으로 바꿉니다.  
5. 각 프레임 **최상위 배경** → Fill `#0D0D0D`, 카드·리스트 영역은 `design/dabonyang-figma-dark-light-tokens.json` 의 `themes.dark` 값으로 맞춥니다.  
6. **텍스트**는 primary `#F5F5F5`, secondary `#A3A3A3`, CTA/강조는 액센트 `#E82127` 유지.

### 라이트 “세련” 버전 (기존 틸 대체)
- 같은 방식으로 **위쪽** 등 여유 캔버스에 복제하고 `(다보냥 · Light)` 접두사를 붙입니다.  
- `themes.light_refined` 토큰으로 배경·카드·구분선을 **회색 스케일 + 레드 포인트**로 통일합니다.

### 토큰 파일
- **`design/dabonyang-figma-dark-light-tokens.json`** — 라이트/다크 Fill·Stroke·텍스트 역할별 HEX. MCP로 색을 입힐 때 또는 로컬 스타일 생성 시 그대로 복사해 쓰면 됩니다.

### MCP로 자동화하고 싶을 때 (연결된 경우)
- 채널 조인 후, 플러그인에서 복제한 노드 ID를 알면 `set_fill` / 텍스트 색 클래스 등 **패키지가 제공하는 도구**로 일괄 적용이 가능합니다. 도구 이름은 패키지 버전마다 다를 수 있어, 연결 후 Cursor가 노출하는 Talk to Figma 도구 목록을 기준으로 쓰면 됩니다.
